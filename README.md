# InicioMySQL-NodeJS
Repositorio de una App hecha con Node js, ejs, Bootstrap y MySQL.

App CRUD de noticias en formato tabla.

Paquetes utilizados:
* express
* body-parser
* mysql
* ejs
* method_override
